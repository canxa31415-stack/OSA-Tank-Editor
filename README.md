# OSA-Tank-Editor
A tank editor/tank viewer for OSA/APS++

## I made this in WebStorm, I cant garuntee this will work elsewhere
WebStorm download: https://www.jetbrains.com/webstorm/

Instructions:
- Download zip file
- Unzip it
- Open the folder as a WebStorm project
- Go to `index.html` and run it
- If that doesnt work try typing `npm i ws` into the terminal

## Disclaimer
"for loops" are not supported

tanks in the code input must be seperated by `,`s
